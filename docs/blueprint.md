# **App Name**: WisdomWell

## Core Features:

- Homepage with Wisdom Feed: Displays a curated feed of popular wisdom articles in a card-based layout, emphasizing recent and trending posts.
- Article Detail View: Presents wisdom articles with adjustable text sizes and displays author profiles including pre-retirement career information, enhancing readability and author credibility.
- Writing Mode with Auto-Save: Offers a 'focus mode' editor with auto-save functionality and prominent placement of an AI sentence refinement tool to simplify content creation.
- Category-Based Article Discovery: Enables users to browse content by category using an icon-driven navigation for easy access to specific knowledge domains.
- Simplified Social Login: Streamlines user access with one-click login options via Kakao or Google.

## Style Guidelines:

- Primary color: Deep teal (#276678) to inspire trust and comfort.
- Background color: Soft warm white (#F9F7F2) to minimize eye strain.
- Accent color: Orange (#E67E22) to highlight key actions like writing and submitting content.
- Text color: Slightly muted dark gray (#2C2C2C) to reduce eye fatigue.
- Body and headline font: 'Pretendard' sans-serif for comfortable reading.
- Font size: Base font size of 18px-20px for improved readability.
- Line spacing: Generous line-height of 1.8 for clear separation of text lines.
- Emphasize clear, large buttons and icons accompanied by descriptive text.
